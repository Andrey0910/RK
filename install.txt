1 Достаем проект и git перейдя в папку куда надо стянуить проект и выполняем команду git clone 'url - к проекту'
2 Устанавливаем используемые библиотеуи в проекте. В консоле перейти в каталог проекта и выполнить
команду php composer install.
3 Правим файл .env. Прписываем настройки для базыданных: DB_HOST, DB_DATABASE, DB_USERNAME, DB_PASSWORD.
Добавить следующие настройки:
- FILE_IMG_PATH //Назвпние папки для фоток относительно корня проекта (паки public)
- PROFILE_FILE_IMG_PATH=profile //Название папки дла фоток в профиля пользователя
- FIT_FILE_IMG_PATH=fil_207 //Название папки уменьшенных фоток для галерей
4 Создание базы данныз. Настройкм к базеданныз ужедолжны быть прописаны в файле .env (пункт 3). В консоле назодясь
в папке проекта выполните каманду php artisan migrate:install.
5 Заполнение базы данных нужной информацией. В консоле назодясь в папке проекта выполните
каманду php artisan db:seed.

Все проект готов к работе. Для его запуска локально надо в консоле пкркйти в паку проекта и выполнить
команду php artisan serv. После этого в браузере можно набрать localhost:8000. Проект запущен.

Для отмены миграций и заново создании базы данных с наподнением данных можно выполнить
укоманду php artisan migrate:regresh --seed (эта команда полностью пересоздаст базу данныз с наполненными данными)
